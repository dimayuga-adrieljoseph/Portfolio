# 📌 AI Agent Prompt: Vue.js Portfolio Website Generator

## 🎯 Objective

Create a fully functional, modern, and responsive personal portfolio website using **Vue.js** (preferably Vue 3 with Composition API and Vite).

---

## 📝 Instructions

Build a complete portfolio website with a clean, modern UI/UX design. The design should follow current web trends:

- Minimalist layout
- Smooth animations
- Good spacing and typography
- Soft shadows
- Fully responsive design

---

## 🧱 Tech Requirements

- Use **Vue.js 3 + Vite**
- Use **Vue Router** for navigation
- Use **Pinia** for state management (if needed)
- Use **Tailwind CSS** (preferred)
- Use **Font Awesome** or similar for icons
- Implement **Dark Mode**
- Ensure full **Responsiveness** (mobile, tablet, desktop)

---

## 📄 Required Pages

### 1. Home Page

- Hero section (name, title, short intro, CTA buttons)
- Profile image or illustration
- Quick highlights (skills, experience, projects)

### 2. About Page

- Detailed personal introduction
- Skills section (icons or progress bars)
- Timeline (education, experience)

### 3. Projects Page

- Grid or card layout of projects
- Each project should include:
  - Image/thumbnail
  - Title
  - Description
  - Tech stack
  - Links (GitHub, Live Demo)

### 4. Contact Page

- Contact form (name, email, message)
- Social media links
- Email integration-ready (API placeholder)

### 5. Resume Page (Optional but Recommended)

- Downloadable resume button
- Inline preview or structured experience list

---

## 🧩 Components to Create

- Navbar (sticky, responsive, mobile hamburger menu)
- Footer (social links, copyright)
- Project Card Component
- Skill Badge Component
- Timeline Component
- Contact Form Component
- Dark Mode Toggle

---

## 🎨 Design Guidelines

- Use a **modern color palette** (neutral + accent)
- Apply **smooth animations** (hover effects, transitions, page fades)
- Maintain **consistent spacing and typography**
- Use **rounded corners and soft shadows**
- Optionally use **glassmorphism** or subtle gradients

---

## ⚙️ Features

- Dark/Light mode toggle with persistence
- Smooth scrolling navigation
- SEO-friendly structure (meta tags)
- Lazy loading for images
- Reusable components
- Clean and scalable architecture
